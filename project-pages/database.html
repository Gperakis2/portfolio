Technical University Database
Implementation of a database for a polytechnic department.
Project Details
Database Description: The database contains data related to the department’s operation, following given specifications. An ER diagram, PostgreSQL relational database implementation, and initial data are provided.
Data Management: Functions for inserting personal data, grades, proposed enrollments, and semester courses are to be created in PostgreSQL.
Data Retrieval and Calculations: Functions for retrieving data and calculating grades for students, courses, laboratories, and sectors are to be developed in PostgreSQL.
Java Application: Use of Eclipse and JDBC to connect to a PostgreSQL server and offer a menu of options for various operations, such as inserting, updating, deleting, and displaying data from the database. The application also handles transactions and exceptions.
Updatable Views: The project requires creating an updatable view on one of the tables of the database that is considered useful and testing its functionality. It also requires creating a complex view that shows all the labs of the department with their directors, contact emails, titles, fields, and sectors.
